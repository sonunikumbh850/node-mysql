# Node.js MySQL CRUD Example

This project is a **beginner-friendly Node.js application** that demonstrates how to connect **Node.js with MySQL** and perform basic **CRUD operations** (Create, Read, Update, Delete).

It is designed for learning purposes and keeps everything simple and easy to understand.
